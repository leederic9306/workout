# 근력 운동 트래커 앱

## 서비스 개요

근력 운동을 체계적으로 진행하고자 하는 사용자에게 검증된 운동 프로그램, 800+ 운동 도감, 개인 운동 기록 관리 기능을 제공하는 Android 전용 모바일 애플리케이션. 헬스장 기구 가용성에 따라 대체 운동을 즉시 전환할 수 있는 것이 핵심 차별점이다.

---

## 타겟 사용자

- **수준**: 초보자 ~ 중급자 근력 운동 사용자
- **니즈**: 체계적인 운동 프로그램을 따르며 진행 상황을 기록하고 싶은 사용자
- **환경**: 헬스장, 홈짐 등 다양한 운동 환경에서 유연하게 운동하는 사용자
- **초기 규모**: 약 5명 (개인 및 지인)
- **플랫폼**: Android 전용 (iOS 미지원)
- **배포 방식**: EAS Build APK 직접 배포 (Google Play 미사용)

---

## 핵심 가치 제안

- **검증된 프로그램 제공**: 5x5 풀바디, 선형 진행, PPL, Arnold Split 등 6종 카탈로그
- **AI 기반 개인 맞춤 추천**: Claude Haiku 4.5를 활용한 맞춤 프로그램 생성 및 재평가
- **기구 제약 해결**: 부재 기구에 대해 태그 기반 점수 산출로 대체 운동 자동 추천
- **개인화된 무게 추천**: 1RM 기반 자동 무게 세팅 (Epley + Brzycki 공식 평균)
- **기록 기반 성장 추적**: 1RM, 체성분, 볼륨 변화 시각화 대시보드

---

## 핵심 기능

1. **사용자 인증 및 권한 관리**: 초대 코드 기반 가입 제한, 이메일/소셜(Kakao, Google, Apple) 로그인, RBAC(Admin/Premium/User)
2. **운동 도감 (Exercise Library)**: 800+ 운동, 부위·기구·난이도별 필터, 시연 이미지/GIF, 주동근·협력근 정보
3. **1RM 관리**: 5종 컴파운드 운동 직접 입력 및 Epley/Brzycki 공식 추정, RPE 기반 자동 추정
4. **운동 프로그램**: 6종 카탈로그 제공, AI 맞춤 프로그램 생성(Premium/Admin), 활성 프로그램 1개 제한
5. **운동 세션 기록**: 세트별 무게/횟수/RPE/휴식 시간 기록, 휴식 타이머, 플레이트 계산기
6. **대체 운동 추천**: 기구 부재 시 태그 기반 유사도 점수로 즉시 대체 운동 제안
7. **마이페이지 / 대시보드**: 1RM 이력, 체성분 이력, 주간 볼륨, 운동 빈도 차트 (4종)
8. **AI 추천 기능**: 카탈로그 추천(월 5회), 맞춤 프로그램 생성(월 10회), 프로그램 재평가(월 10회)
9. **푸시 알림**: 일일 운동 알림, 7일/14일/30일 공백 알림 (Expo Push)
10. **온보딩**: 닉네임, 성별, 생년월일, 키, 운동 경력 수집

## 사용자 권한 체계

| 역할 | 주요 기능 |
|------|-----------|
| **User** | 운동 도감, 카탈로그 프로그램, 운동 세션, 기록, 대시보드, 알림, AI 카탈로그 추천 (월 5회) |
| **Premium** | User 기능 전체 + AI 맞춤 프로그램 생성 (월 10회) + AI 프로그램 재평가 (월 10회) |
| **Admin** | Premium 기능 전체 + 사용자 관리 + 초대 코드 관리 + AI 사용량 모니터링 |

- JWT Access Token 유효 기간: 15분
- JWT Refresh Token 유효 기간: 30일 (Rotation 적용)
